### 기능 목록
- [ ] 입력받은 자동차 이름을 쉼표 기준으로 구분한다.
- [ ] 이름의 길이가 5자를 넘지 않는지 검증한다.
- [ ] 시도할 횟수를 입력받는다.
- [ ] 무작위 값을 구한 후 이동한다.
- [ ] 우승자를 찾아낸다.
- [ ] 우승자가 몇 명인지에 따라 안내 문구를 출력한다.

### 테스트 목록
- [ ] 입력받은 자동차 이름 구분 테스트
- [ ] 이름의 길이 검증 테스트
- [ ] 시도할 횟수 입력 테스트
- [ ] 무작위 값 이동 테스트
- [ ] 단독 우승자 확인 테스트
- [ ] 공동 우승자 확인 테스트
